# 🛡️ CyberSense-CSE445  
### Cyberbullying Detection System with LLM Chatbot

An AI-powered cyberbullying detection platform that classifies harmful online text using Machine Learning and Deep Learning models, integrated with a web interface and LLM-based chatbot for user guidance.

---

## 📌 Project Overview

Cyberbullying is a growing issue across social media, gaming platforms, and online communities. This project aims to automatically detect cyberbullying text using Natural Language Processing (NLP) techniques and predictive models.

The system allows users to:

- Enter text through a website interface
- Detect whether content is cyberbullying or not
- Receive chatbot-based explanation/guidance

---

## 🚀 Features

- Binary Cyberbullying Classification
- Advanced NLP Text Preprocessing
- TF-IDF + TextBlob Feature Engineering
- GloVe Embeddings for Deep Learning
